---
## Описание
Этот проект представляет собой простую игру "Разбей блоки", написанную на Python
## Установка 
- требуется Python 3
- У вас должна быть установлена библиотека pygame
## Правила 
Их нет!

- при проигрыше следуйте указаниям на экране
